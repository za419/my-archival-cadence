About Cadence:

Cadence is an online radio whose creation was inspired by the anime-themed R/a/dio. Originally started as an experiment in February 2017, the project is my first endeavour to practice a full range of IT-skills.

Cadence's frontend is served through a Node.js webserver interacting with a Mongo music database, both configured around a Liquidsoap/Icecast stream server.

The server typically plays all the genres of music I like, a mix of classic rock, synthpop, and metal. I occasionally override the server and play a certain genre, artist, or the same song on an endless loop.

If you're interested in seeing this project's development, nearly everything is available on GitHub: source/issues.

Cadence Radio is a DMCA compliant, "non-commercial educational" (NCE) broadcast. As an NCE, Cadence Radio is non-profit and does not accept advertisements for its webpages or broadcasts.

Announcements

5/1: Fermata

On Saturday, May 5th, I am moving out of my dorm, and Cadence's server will be taken down.

I can't provide an accurate time estimate for when it will return. This summer, I intend to find a new place to host, as well as make new server hardware purchases.

This year has seen incredible new features and improvements. The installation of new server software allowed us to create song requests functionality in the ARIA project, finishing one of our toughest goals. Half a dozen new themes were provided by Cadence contributors as well. Though an issue dealing with database indexes still exists, I've decided to let this one go for now and ceremonially declare Cadence to finally have hit version 2.0 before we go down for the summer.

If you are interested in donating, I have a personal PayPal account here. If you still want a Cadence sticker, I still have some left available for cheap purchase(contributors receive one for free).

That said, thank you all again for listening to Cadence Radio. Special thank you to our contributors:
